* # HEAP_PAIR: heap-pair class for C++ ![license-status](https://img.shields.io/github/license/Dark-CodeX/heap-pair)
	* #### **heap-pair class in C++, for storing values in heap memory.**
	* #### This header file is *cross-platform*, but tested only on Windows 10, Windows 11, Arch Linux, Debian Linux, WSL 2 only. (But will work on macOS also).
